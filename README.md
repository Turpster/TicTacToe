# TicTacToe

Just the tic-tac-toe multiplayer game with dynamic game size (3x3, 4x4, 5x5).
</br>

It uses:
* WPF 
* MahApps.Metro UI Toolkit
* Autofac IoC container
* MVVM pattern
* MVVM Light Toolkit
